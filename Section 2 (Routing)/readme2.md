## Metadata API in Next.js (s2 : ep7)

- how to set metadata of next js using metadata api.
- jo body de ander hota hai vo dikhta hai pr jo body ke ander nhi hota meta data head ke ander.
- we dont manually add head tag.
- we can use metadata api :
  - we export an object named metadata

```js
export const metadata = {
  title: "Create Next App",
  description: "Generated by create next app",// used for seo
};
```

- if this code on root layout then it will be same for every route
- i can also add it on other pages of routes.

```js

page.js inside root app folder

export const metadata = {
  title: "Home | Create Next App",
  description: "Generated by create next app",
};


page.js inside about folder

export const metadata = {
  title: "About | Next App",
  description: "Generated by create next app",
};

```

- we can also set template for this object in root layout.js.

```js

export const metadata = {
    title:{
        template:"%s | Technical agency"
        default: "Next App",
    }
  description: "Generated by create next app",
};

```

now here in

- template : %s we will recieve the title of other metadata objects.
- default : yhan basically jo value di hogi vo home page pe show hogi.


### generating dynamic title for every page.

```js 

export async function generateMetadata({params}){
    
    const {blogId} = await params;
    
    return {
        absolute:`Blog ${blogID}`
    }
}

```
